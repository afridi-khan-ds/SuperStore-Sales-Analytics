# SuperStore Sales Analytics & Forecasting

## Overview
This project delivers an end-to-end analysis of SuperStore sales data, combining historical sales analysis with short-term sales forecasting to generate actionable business insights. The workflow includes data cleaning, SQL analysis, and interactive Power BI dashboards to support data-driven decision-making.

## Dataset
Dataset Name: SuperStore_Sales_Dataset  

The dataset contains transactional sales records including order dates, sales, profit, regions, product categories, customer segments, shipping modes, and payment methods.

## Tools & Technologies
- Python for data cleaning and exploratory analysis  
- SQL for querying and aggregating data  
- Power BI for interactive dashboards and forecasting  
- DAX for KPIs and calculated measures  

## Project Objectives
- Analyze sales, profit, and order performance  
- Identify top-performing regions, states, and products  
- Understand customer segments and shipping behavior  
- Track monthly and year-over-year trends  
- Forecast short-term sales for business planning  

## Dashboards
Sales Performance Dashboard  
Provides a consolidated view of key metrics such as total sales, profit, orders, and delivery time with breakdowns by region, category, segment, and shipping mode.
Screenshots are included in the screenshots folder for quick preview.

Sales Forecast Dashboard  
Forecasts sales for the next 15 days using historical daily sales data and Power BI time-series forecasting with confidence intervals.

## Forecasting Methodology
- Daily sales aggregated by order date  
- Power BI built-in forecasting model  
- 15-day forecast horizon with confidence bands  
- Dynamic interaction with filters and slicers  

## Business Value
- Enables data-driven business decisions  
- Supports short-term sales and inventory planning  
- Highlights performance gaps and growth opportunities
